Automatic login settings

When travian logs out, extension can login to continue doing its work. Your username and password will be stored in extensions storage locally without syncing it between devices. You can delete stored data at any time.

Attack notification settings

Extension will refresh page at random time in the given range. It's necessary because of hiding from BOT hunter. Defaults to 2-5 and values are given in minutes. To turn feature off, add 0-0.
